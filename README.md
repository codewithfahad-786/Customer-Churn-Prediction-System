# 📊 Customer Churn Prediction System

An end-to-end Machine Learning web application that predicts whether a customer will churn (leave) or stay, and provides risk level classification using a trained ML model.

---

## 🚀 Project Overview

This project helps subscription-based businesses (telecom, SaaS, streaming) to:

- Predict customer churn probability
- Identify high-risk customers
- Support retention decision-making
- Provide a user-friendly web interface

---

## 🧠 Problem Statement

Companies lose customers every month but cannot identify who is likely to leave.  
This system predicts churn using machine learning and helps businesses take proactive actions.
